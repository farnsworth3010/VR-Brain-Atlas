# VR Brain Atlas

3D VR-просмотрщик медицинских моделей мозга: мозг, опухоли, сосуды и костный слой. Приложение позволяет вращать и перемещать модели, менять масштаб и включать/выключать слои для изучения анатомии в виртуальной реальности.

## Основные возможности

- Просмотр 3D-моделей мозга, опухолей, сосудов и костей
- Вращение, перемещение и масштабирование моделей
- Включение / отключение слоёв (brain / tumor / arteria / skull)
- Поддержка загрузки моделей из папки или через диалог выбора файлов
- Автоматическое позиционирование моделей из `model_data.json`
- Викторина по анатомии мозга (тест на знание структур)
- Интерактивная викторина по выбору точки разреза опухоли

## Требования

- Unity 6
- .NET / C#
- Git и Git LFS для загрузки больших ассетов

## Установка (через Git)

1. Установите Git LFS, если он ещё не установлен:

```bash
git lfs install
```

2. Клонируйте репозиторий и загрузите большие файлы:

```bash
git clone https://github.com/your-username/vr-brain-atlas.git
cd "vr-brain-atlas"
git lfs pull
```

## Запуск проекта

- Откройте проект в Unity Editor (через Unity Hub или напрямую, указав папку проекта).
- Подождите, пока Unity импортирует ассеты и соберёт проект.
- Откройте нужную сцену (в списке `Assets/Scenes` или через окно сцены) и нажмите Play.

## Поддерживаемое оборудование

- HTC Vive (через SteamVR)
- IVRY (через SteamVR)
- Телефон в режиме VR (через совместимые мосты/плагины SteamVR)
- Управление: геймпад Xbox или другие совместимые контроллеры

> Для корректной работы в VR убедитесь, что установлены и включены соответствующие драйверы (SteamVR или провайдер вашего устройства) и что в Unity настроен XR-плагин для вашей платформы.

## Загрузка и использование 3D-моделей

Подготовьте модели из МРТ-снимков с помощью [swiss_skull](https://github.com/AlekseyShugol/swiss_skull) — программа преобразует МРТ в 3D-модели мозга и опухоли и автоматически создаёт `model_data.json`.

Поместите экспортированные файлы в папку `Documents/VR Brain Atlas` или выберите файлы вручную через диалог в приложении.

**Важно:** файлы моделей должны называться строго `brain.obj`, `tumor.obj`, `arteria.obj` или `skull.obj`.

> В папке [`Examples/`](Examples/) репозитория находятся пример моделей (`brain.obj`, `tumor.obj`) и готовый `model_data.json` с заполненной секцией `incision_quiz`. Скопируйте содержимое папки в `Documents/VR Brain Atlas` для быстрого тестирования.

## Файл model_data.json

При наличии файла `Documents/VR Brain Atlas/model_data.json` приложение автоматически считывает позиции моделей и данные для викторины разреза.

### Пример файла

```json
{
  "models": {
    "brain": {
      "position": [0, 0, 0]
    },
    "tumor": {
      "position": [30, 7, 9.2]
    },
    "skull": {
      "position": [-0.36, 66.68, -7.77]
    },
    "arteria": {
      "status": "failed"
    }
  },
  "coordinate_system": "unity",
  "units": "millimeters",
  "incision_quiz": {
    "point_radius_mm": 30.0,
    "correct_point_index": 1,
    "correct_explanation": "Латеральный доступ слева минимизирует повреждение функциональных зон коры.",
    "wrong_explanation": "Эта точка проходит через критические структуры или требует значительно большего пути.",
    "points": [
      {
        "index": 0,
        "label": "+X (правый латеральный)",
        "direction": [1, 0, 0],
        "position": [-6.4, 79.6, -30.7],
        "is_correct": false,
        "note": "Правое полушарие — опухоль в левом, доступ неоправданно длинный"
      },
      {
        "index": 1,
        "label": "−X (левый латеральный)",
        "direction": [-1, 0, 0],
        "position": [5.0, 103.2, 1.7],
        "is_correct": true,
        "note": "Оптимальный доступ: кратчайший путь к опухоли через левую теменную кору"
      }
    ]
  }
}
```

### Поля модели

| Поле | Описание |
|------|----------|
| `models.<name>.position` | Позиция модели в миллиметрах (система координат Unity). Применяется с множителем `0.05`. |
| `models.<name>.status` | `"ready"` или `"failed"` — модели со статусом `"failed"` пропускаются. |
| `incision_quiz` | Секция с данными для викторины разреза (опциональна). |

## Викторина по анатомии

Стандартный тест с вопросами и двумя вариантами ответа. Вопросы загружаются из файла `Assets/Resources/quiz.json`.

После каждого ответа отображается:
- Правильность выбора (✓ / ✗)
- Правильный ответ и пояснение
- Итоговый счёт в конце

## Викторина по выбору точки разреза

Интерактивный режим, доступный при наличии секции `incision_quiz` в `model_data.json`.

После загрузки моделей появляется кнопка **«Начать викторину»**. При запуске:

1. На поверхности опухоли появляются сферы-маркеры (точки возможного разреза).
2. Наведите луч контроллера или геймпада на точку — она подсветится жёлтым.
3. Нажмите кнопку выбора — отобразится результат:
   - **Зелёная точка** — правильный вариант разреза.
   - **Красная точка** — выбранный неправильный вариант.
